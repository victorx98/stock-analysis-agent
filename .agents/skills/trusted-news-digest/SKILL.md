# Trusted News Digest

## Procedure

1. Read `rules/02-source-quality.md` and `rules/04-news-and-catalyst-analysis.md`.
2. Classify each source by tier.
3. Group news by catalyst category.
4. Apply the materiality test before including an item in the final brief.
5. Separate confirmed facts from speculation.
6. Write both bullish and bearish interpretations for material items.

## Output checklist

- Material news table.
- Source tier and URL/path for each item.
- Catalyst category.
- Time horizon.
- Bullish interpretation.
- Bearish interpretation.
- Follow-up questions.
